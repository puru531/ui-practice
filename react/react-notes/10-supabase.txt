----------------------- What is Supabase -------------------------
> It is a service that allows developers to easily create a back-end with a Postgres database.
> Automatically creates a database and API so we can easily requst and receive data from the servver.
> Noo back-end development needed.
> Perfect to get up and running quickly
> Not just an API : Supabase also comes with easy-to-use user authentication and file storage.
