Buffers the source Observable values for a specific time period.

Collects values from the past as an array, and emits those arrays periodically in time.

bufferTime(bufferTimeSpan: number, ...otherArgs: any[])

bufferTimeSpan - The amount of time to fill each buffer array.
Others configuration arguments such as:
bufferCreationInterval - the interval at which to start new buffers;
maxBufferSize - the maximum buffer size;
scheduler - the scheduler on which to schedule the intervals that determine buffer boundaries.
