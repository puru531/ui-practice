Returns an Observable that skips the first count items emitted by the source Observable.

Skips the values until the sent notifications are equal or less than provided skip count. It raises an error if skip count is equal or more than the actual number of emits and source raises an error

Parameters: count - The number of times, items emitted by source Observable should be skipped.

Returns: A function that returns an Observable that skips the first count values emitted by the source Observable.
