Unlike Subject, (which doesn't take any default value as the default parameter),
Behavior Subject takes the dafault value while declaring the subject.
Also it stores the last emitted value.

let behaviourSubject$ new behaviourSubject(0)
It will take 0 as the initial value.

behaviourSubject sends the initial Value to the observer when subscribed before emitting the value.
After value is emitted, the behaviourSubject sneds the latest value to all the observers.
